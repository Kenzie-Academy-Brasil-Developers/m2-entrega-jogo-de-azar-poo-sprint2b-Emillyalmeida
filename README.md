
JokenpôJs
=======

Introdução
----------

Criação de um jogo classico usando os conceitos de
**POO** aprendidos.

------------------------------------------------------------------------

Pedra, Papel, Tesoura
---------------------

O jogo clássico onde duas pessoas escolhem entre pedra, papel ou
tesoura, e mostram suas escolhas com as mãos.

Regras:

-   Escolha entre pedra, papel ou tesoura;


**Lembre-se!**

* Pedra vence tesoura;

* Papel vence pedra;

* Tesoura vence papel;
------------------------------------------------------------------------
